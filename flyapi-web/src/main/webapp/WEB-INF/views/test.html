嗒哒木

跟大家分享一下前些日子看的电影，它的名字叫：后会无期。至于为什么去看，我想说几点我的观点。

关于导演

有人问：《后会无期》是一个公路片？文艺片？还是？我会说：都不是，《后会无期》是一个韩寒导演的一部片子。对于韩寒的感情，还要从他的17岁说起。那时候他只是一个高中辍学了初出茅庐的一个小孩子。但是，他对于我们这一代年轻人来说，更像是一个符号的存在。他能实现我们不能实现的关于理想关于叛逆的梦。

关于剧情

好看的书千千万，韩寒的书也是很好看的，好看的电影自然也是千千万。对于韩寒处理电影来说，我并没有报以很大的期待。毕竟不是科班出身，也是第一次尝试将文字搬上大银幕。我愿意为他贡献这一份票房，支持他继续电影的创作。好看是个很主观的词汇，我只能说说我所认为的好看。我喜爱的韩寒就是我要去看的理由，剧情的好看也是建立在自己的主观意识之上的。

一些细节


对于故事的叙述，不仅仅是语言和文字，也有在配乐上的功夫需要下。后会无期用了两首足以打动自己的歌曲，一首《平凡之路》一首《后会无期》。朴树时隔11年为其创作的歌，无疑是打动到了很多热爱民谣的孩子了。也是深深的打动到我。而邓紫棋的翻唱，也是恰到好处的引起了我的注意，这也是我喜欢一部电影的原因——音乐。音乐是魂是感情所在。

总结

如此种种，喜欢韩寒的人可能仍然会一直喜欢，喜欢所有的一切。

派网友

通常，奢侈品牌定价考虑的主要因素有三个

一是基于市场需求考虑的利润最大化，

二是基于竞争策略考虑的利润最大化，

三是基本成本因素考虑的利润最大化。

现在，奢侈品牌千方百计地让公众看到的是第三个因素由于高额的成本所带来的合理利润，而屏蔽了前两个因素，而在成本因素中，制造成本、人员成本、营销成本、渠道成本都属于品牌自身掌握的信息，而只有税率这个环节是透明的，是外部的，中国高关税带来的价差就成了一个很好的提价理由。


其实，奢侈品的进口关税在很多品牌整体成本比例中的占比并不大，占市场定价的比例也不大，不足以支撑如此大的价差，奢侈品牌往往通过各种形式和渠道“合理”避税，如原材料进口、半成品进口甚至是直接在国内生产。事实上，我国政府近年来也在不断调低高端进口商品的进口关税。


更掩人耳目地，奢侈品牌的产品在中国的到岸价已经被人为高估，特别是奢侈品牌在中国代工已经不是秘密，李鬼摇身变李逵，实际制造成本与到欧洲贴标后的价格差就已经产生了高额利润，而品牌巧妙地把这部分利润转嫁到各个环节中去，而被“黑”了的中国代工厂们，只能辛辛苦苦地赚几个血汗钱，还诚惶诚恐。


所以，关税只是奢侈品牌的借口，品牌有意在利用关税掩盖其暴利的行径，并以关税为借口，提高自己在产业链的各个环节的溢价能力，以获得更多更好的商业合作条件。

又是晴天的阴天

我认为心理学应该算是一种科学，最起码从我最初接触心理学时，就理所应当的认为它是一门科学。不然为什么我们要去学习它和研究它呢？

心理学介绍

心理学包括基础心理学与应用心理学两大领域，其研究涉及知觉、认知、情绪、思维、人格、行为习惯、人际关系、社会关系等许多领域，也与日常生活的许多领域——家庭、教育、健康、社会等发生关联。心理学一方面尝试用大脑运作来解释个体基本的行为与心理机能，同时，心理学也尝试解释个体心理机能在社会行为与社会动力中的角色。

心理学分析

既然心理学也属于我们人类需要研究和分析的学科，那么它理应是一种科学。科学本身就是深不可测，需要人类不断的探索。不断地在探索中进步与学习，又不断的进行更深入的探索。如此循环往复的进行下去。人类需要不断的去学习，去利用科学探索的结果。而心理学恰恰与上述观点都相符，怎么又会被人质疑他作为科学所存在的依据呢。

心理学是科学的依据

第一，它深不可测，需要我们人类不断去探索。第二，它是一门开放的学科，并且被公认。第三，它造福于人类。现在大学里有主修心理学的专业。而且很多心理上有疾病的病人都会去看心理医生。心理医生又拯救了大批精神病患者。然而心理学医生是通过对心理学的学习来造福人类。所以我们应该坚信心理学是一门科学，

心理学在我们的日常生活工作中发挥着十分重要的作用。

差点一米六

现在喊自己老了的人一般都是90后，因为到了2018年，00后成年了所以最近很多人都在网上喊自己老了。

我觉得这很正常，老不是体现在年龄上，有些人二十几岁心理年龄好似七老八十一样，对生活失去兴趣。83岁的设计师卡尔，听到他的年龄会认为他是个生命力衰弱的老头子，看到本人却只让你脑海浮现神采奕奕这四个字，状态特别好。

缺乏生产力

二十几岁的这一代年轻人正是生活刚富裕的时候，都是被父母爷爷奶奶宠在手心里的，所以从小经历的事少，毕业了需要靠自己在社会上打拼立足，处于二十几岁的年轻人成为这个世界的生产者，刚从象牙塔一样的大学毕业又变成世界上忙碌的生产者，这一身份改变的过快，忙碌劳累让心理年龄变得衰老，因此很多年轻人觉得自己老了。

失去好奇心

人到了二十几岁心智都成熟了，无论是学习、工作、生活状态压力都比较大，对生活的热情削减了，也不再有对万物的好奇渐渐消失，没有了探索美妙生活的勇气，不愿意改变死气沉沉的生活态度，所以容易产生自己老了这种错觉。在生活中我们要保持一颗好奇心，它能带给我们很多好处 。

所以我们要改变自己消极的心态，不要总是嘴上喊老，我们年轻着呢！

先不说内存的事，你确定你要一台手机用四年吗？iPhoneX的掉漆门你确定能用四年吗？买得起iPhoneX干嘛要用四年呢，不应该一年一换吗？首先就从一个手机的外观磨损来说，先不说这次iPhoneX爆出的掉漆门，就正常磨损来说用个一年手机的划痕就非常明显了，用个两年那基本没法看，所以你如果一台手机要用四年的话，首先要忍受的是外观磨损。

第二个就是系统的问题了，虽然说用个两三年IOS不至于卡到用不了的程度，但是身边的例子告诉我基本上到了第三年反应就会有点跟不上了，游戏也不能开高特效了、打开APP的速度也慢了不少，程序无响应的情况也会时有发生，这也是需要你忍受的。

四年后的iPhoneX怎么样，你问一下你身边用iPhone5S或者用iPhone6的人就知道了。

还有一点你需要注意的就是苹果的电池在使用两年之后容量是急剧衰减的，基本上你一天要充个三到四次电了，苹果手机的电池容量本身就小，一衰减就更加捉襟见肘了，唯一的办法你就要拆机换电池了，在外面换的话可能手机就不防水了，在售后换的那估计又要大几百。

然后再来说说内存的事，如果你是想用四年还不把资料往电脑或者U盘里面放的人，那肯定是妥妥的256G版本的，说实话就我个人来说的话平时拍照少、看视频都是在线、歌曲也基本用流量在线听，32G的内存就足够我用了，现在是已经普及了64G了，我现在64G的一半都没用完，而且我个人还有习惯把照片往电脑里面备份的习惯。

如果你的习惯和我一样那我建议你买64G的，毕竟差了1300块钱这里啊，1300块钱拿来买点吃的买件衣服多好啊！话说回来你买256G的也不划算，你要真的不差钱就不会iPhoneX用四年了。

梵蒂冈

心理咨询中咨询费用是最高的那个门槛。

社会现象

对于生活压力小的人，心理疏导的原因很多，如心态好，抗压能力强，时间不够，接受不够，找不到合适的咨询师等等，来承担按揭车贷款，没有房子，没有车，担心基本的日常饮食的群体来说，首当其冲的原因是：”好贵啊！“1000元/小时，800元/小时，500元/小时 ，可能还有更贵的。心理问题大到必须“咬牙支出额外费用去做心理咨询”才能解决的情况，太少了！

观念问题

很多去咨询的人会说：“我又没生病，干嘛要做心理咨询”或者“让我们谈谈吧，但不要太正式”，而且咨询师角度看，真正有本事的咨询师太少，少量有本事的咨询师搞培训带徒弟的收入来的快的多，真正要找他做咨询，一是没时间，二是奇贵。心理有病忍忍就好了，打打牌喝喝酒逛逛街做做美容泡泡妞看看电影骂骂人就转移注意力了。从心理的角度看，在中国社会，人拒绝成长可以获得额外好处，比如可以继续啃老，可以让爱人疼自己，可以少干活，等等，治疗好了反而要自己去面对生活，还要承担起很多责任，很多人没有这种勇气。“我不想我不想长大”其实是很多中国人的心声。

总结：

想要心理学壮大，除非变得比打牌还好玩比按摩还舒服，而这确实是可以实现的。加油吧！

臧美然

小米的解决方案是最暴力的。就是想方设法想把边框做没…

iPhoneX依然选择了异形全面屏这条路，其目的就是做出当下最接近人们心中的全面屏的状态：四周无边框，整个手机正面就是一块玻璃。不过由前文所知，即便采用COF，仍然需要留一块地方留给软性电路板。因此可以看到，如今的技术已经可以支持手机做出三面无边框的设计，而最关键的那一部分，则不再仅仅是结构上的问题，而需要通过屏幕材质来解决。因此我们才看到，苹果在iPhoneX上使用了AMOLED屏幕，以达到所需要的效果。因为柔性OLED屏幕所采用的基板材质与软性电路板的材质都为Pi膜，通过翻卷的方式直接把屏幕下端子区域直接翻卷到屏幕背部。完成全面屏的“最后一公里”。做出真正意义上的全面屏。

而IPhoneX有一个讨巧的地方。MIX2过于追求整机的一体性，将中框颜色尽量融入正面边框，导致“大边框”观感。而IPhoneX采用不同色的金属中框，使人一看便知中框的弧形凸起与边框的区别。因此，乍一看，IPhoneX的边框不比MIX2宽多少。

至于三星，三星确实nb…曲面屏变相解决了全面屏“一半”的问题

首先说一点，苹果的异形屏方案确实可以达到目前技术限制下最大的理论屏占比。如果要比数字的话，肯定是苹果最大。

但是大家也知道异形屏有很多缺陷，那么为什么只有苹果敢用异形屏呢？很简单，因为iOS是他自己的操作系统。所以不管这个屏幕如何的『异形』，软件部门的人总有办法再去修改UI，让各种app合理的进行适配，比如竖屏状态下用这部分空间显示信号和电量。横屏现在虽然被吐槽，但别忘了他们还有2个月的时间完善代码，把横屏的全屏模式缩一下也不过就是几行代码的事情。完全无压力。

相比之下，Android部分的厂商就没那么幸运了，改操作系统UI？想都不要想。Google能允许你们突破16：9已经是格外开恩了好么！所以虽然雷布斯自认为自己的方案很有信心，但如果给修改操作系统UI的权力，估计他还是会拥抱异形屏的……

在这个前提下，S8和MIX就是保守和极端的对比。反映的实际上也是各自品牌的定位，米粉可以不考虑前置摄像头必须倒持使用的各种问题。但三星不可以这么激进，毕竟三星的用户群大多了。那么最终看到的就是现在的样子。

总之，我认为三星 s8、小米 MIX1/2、iPhone X 三种高屏占比手机设计方案各有优劣，大家还是依据自身喜好选择。

古今中外，绿植总是收人深爱。无论是高定走秀，还是展览馆陈列，都一定会有花花草草，绿色的花草总能给人生机勃勃的感觉。绿色植物放在店里有着很多作用。

一、打造丰富的层次感

就像人穿衣服一样，在店铺中布置绿植，不仅丰富了店铺颜色，也增加了整体空间的层次感，且富有生气，简直就是装饰神器。

二、承接室内外的空间

在店内布置绿植，让室内外的空间都连接起来，顾客自外入内才不会出现过大的感受断层。

三、具有指示引导的作用

绿植也可以作为VP点出现在店铺里，是无声的指示牌，出现在出入口、拐角处、死角等地方，吸引顾客的走进，也可以引导顾客的走向。

四、起到间隔、分区的作用

利用绿植对有限的空间加以限定和分隔，使分区更明显，提高空间的利用率。

有哪些提升店铺质感的绿植？越来越多人把绿植搬进店铺里，到底有哪些颜值高、好伺候，还可以提升店铺质感的绿植呢？今天就给大家推荐一下绿植中的网红！！！

多肉植物

作为这些年突然火起来的懒人植物，多肉的颜值那是相当高！其实仙人掌也属于多肉植物的一种，这类植物耐干旱，不需要天天浇水，喜欢阳光充足的地方，但人家也耐阴，夏天可要给人家避避暑~

琴叶榕

琴叶榕，最近兴起的室内观叶植物网红，作为热带植物，大大的绿色叶子就像小提琴一样，具有非常强的观赏性，对空气污染及尘埃抵抗力很强，喜欢阳光充足，温暖湿润的环境。

 散尾葵

又叫富贵椰子，这名字一听就很富贵，所以已经无法形容这货到底有多火了，不瞒你们说，在社长的办公桌旁就有一盆。这类植物太好养了，只要有明亮的光线和通风的环境就能生长，长到一个成人的高度轻轻松松。
